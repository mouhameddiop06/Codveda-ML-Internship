# Level 1 · Task 1 — Data Preprocessing for Machine Learning

**Objective:** Preprocess a raw dataset to make it ready for machine learning.

## Steps covered
- Handle missing data (mean/median imputation or dropping).
- Encode categorical variables (label encoding / one-hot encoding).
- Normalize or standardize numerical features.
- Split the dataset into training and testing sets.

## Dataset
`data/churn_train.csv` — telecom customer churn (has categorical + numerical columns,
ideal for demonstrating every preprocessing step).

## Run
```bash
python data_preprocessing.py
```

## Tools
Python, pandas, scikit-learn.
