# Level 1 · Task 3 — K-Nearest Neighbors (KNN) Classifier

**Objective:** Build a KNN classifier to classify data points into categories.

## Steps covered
- Train a KNN model on a labeled dataset (Iris).
- Evaluate with accuracy, confusion matrix, and precision/recall.
- Try different values of K and compare the results.

## Dataset
`data/iris.csv` — 150 rows, target = **species** (setosa / versicolor / virginica).

## Run
```bash
python knn_classifier.py       # script version
# or open knn_classifier.ipynb
```

## Tools
Python, scikit-learn, pandas, matplotlib.
