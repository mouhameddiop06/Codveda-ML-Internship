# Level 2 · Task 1 — Logistic Regression for Binary Classification

**Objective:** Predict a binary outcome (whether a customer will churn).

## Steps covered
- Load and preprocess the dataset.
- Train a logistic regression model with scikit-learn.
- Interpret model coefficients and odds ratios.
- Evaluate with accuracy, precision, recall, and the ROC curve.

## Dataset
`data/churn_train.csv` (train) and `data/churn_test.csv` (test) — telecom churn.
Target = **Churn** (True / False).

## Run
```bash
python logistic_regression.py
```

## Tools
Python, pandas, scikit-learn, matplotlib.
