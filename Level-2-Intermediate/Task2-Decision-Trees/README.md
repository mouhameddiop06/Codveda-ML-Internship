# Level 2 · Task 2 — Decision Trees for Classification

**Objective:** Build a decision tree classifier to predict a categorical outcome
(species of flowers).

## Steps covered
- Train a decision tree on the Iris dataset.
- Visualize the tree structure.
- Prune the tree (max_depth / ccp_alpha) to prevent overfitting.
- Evaluate with accuracy and F1-score.

## Dataset
`data/iris.csv` — 150 rows, target = **species**.

## Run
```bash
python decision_tree.py
```

## Tools
Python, pandas, scikit-learn, matplotlib.
