# Level 3 · Task 1 — Random Forest Classifier

**Objective:** Implement a Random Forest model for classification on a complex dataset.

## Steps covered
- Train a Random Forest and tune hyperparameters (n_estimators, max_depth) via GridSearchCV.
- Evaluate with cross-validation and classification metrics (precision, recall, F1).
- Perform feature importance analysis.

## Dataset
`data/churn_train.csv` / `data/churn_test.csv` — telecom churn. Target = **Churn**.

## Run
```bash
python random_forest.py
```

## Tools
Python, pandas, scikit-learn, matplotlib.
