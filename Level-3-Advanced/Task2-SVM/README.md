# Level 3 · Task 2 — Support Vector Machine (SVM) for Classification

**Objective:** Implement an SVM model for classification.

## Steps covered
- Train an SVM on a labeled dataset.
- Compare kernels (linear vs RBF).
- Visualize the decision boundary (using two features).
- Evaluate with accuracy, precision, recall, and AUC.

## Dataset
`data/iris.csv`. For a clean binary AUC demo the script keeps two classes
(versicolor vs virginica); the multiclass version is also reported.

## Run
```bash
python svm_classifier.py
```

## Tools
Python, pandas, scikit-learn, matplotlib.
